# 🧠 Brain Tumor Classification using CNN

This project uses a **Convolutional Neural Network (CNN)** to classify brain MRI images into different tumor types. It is a deep learning-based **multi-class image classification model** built using Python and TensorFlow/Keras.

---

## 📌 Project Overview

Brain tumors vary in type and severity, and accurate classification is crucial for diagnosis and treatment planning. This project aims to automate the classification of brain MRI scans into multiple categories using deep learning.

---

## 🎯 Classes

The model classifies MRI images into the following categories:

* **Glioma Tumor**
* **Meningioma Tumor**
* **Pituitary Tumor**
* **No Tumor**

---

## 🚀 Features

* Multi-class image classification using CNN
* Image preprocessing (resizing, normalization)
